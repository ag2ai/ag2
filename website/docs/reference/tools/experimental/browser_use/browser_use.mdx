BrowserUseTool(
    *,
    llm_config: dict[str, typing.Any],
    browser: ForwardRef('Browser') | None = None,

    A class representing a Tool that can be used by an agent for various tasks.
    
    This class encapsulates a tool with a name, description, and an executable function.
    The tool can be registered with a ConversableAgent for use either with an LLM or for direct execution.
    
    Attributes:
        name (str): The name of the tool.
        description (str): The description of the tool.
        func_or_tool (Union[Tool, Callable[..., Any]]): The function or Tool instance to create a Tool from.
    
    Use the browser to perform a task.

<b>Parameters:</b>
| Name | Description |
|--|--|
| `llm_config` | The LLM configuration.<br/><br/>**Type:** `dict[str, typing.Any]` |
| `browser` | The browser to use.<br/><br/>If defined, browser_config must be None<br/><br/>**Type:** `ForwardRef('Browser') \| None`<br/><br/>**Default:** None |
| `agent_kwargs` | Additional keyword arguments to pass to the Agent<br/><br/>**Type:** `dict[str, typing.Any] \| None`<br/><br/>**Default:** None |
